# khaleejbs.com
This is the repository for the website of Khaleejbs.com, a logistics business solutions website built using HTML, CSS and PHPMailer for submitting contact us form details.

# Description
khaleejbs.com is a website that provides information about logistics business solutions to potential clients. The website is built using HTML and CSS and has a contact us form that is built using PHPMailer to submit form details. The website is designed to be responsive and accessible on various devices.

# Installation
To install the website on your local machine, follow the steps below:

1) Clone the repository to your local machine using the command git clone https://github.com/jagadishwar2010/Khaleej.git.
2) Navigate to the project directory using the command cd Khaleej.
3) Open the index.html file in a web browser to view the website.

# Usage
The website is designed to be user-friendly and easy to navigate. Users can browse the website to learn about the logistics business solutions offered by Khaleejbs.com. To contact the company, users can fill out the contact us form and submit their details.
